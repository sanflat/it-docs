### "英文"
* 単語と単語の間は、1スペース。
* 文の最初は、大文字。
* 文の最後は、ピリオド(.)。

example
```
Let's study English.
(英語を勉強しましょう。)
```
* 疑問文の場合は、ピリオド(.)のかわりにクエスチョン・マーク(?)。
```
What's this?
(これは何ですか？)
```
* Yes or Noの場合は、コンマ(,)。
* Iはいつでも大文字
```
Yes, I am.
(はい、そうです。)
```

### "主語" & "動詞"
* 英語の文には、”主語”と”動詞”が必要。
* 主語とは、「私は」「あなたは」「太郎は」のように文の主人公を表す言葉。
* 動詞とは、「走る」「話す」「勉強する」のように動きを表す言葉。

example
```
I play games.
(私はゲームをします。)
```

```
You speak Japanese.
(あなたは日本語を話します。)
```

```
Tarou eat sushi.
(太郎は寿司を食べます。)
```

```
We study English.
(私達は英語を勉強します。)
```


### "be動詞"
* 「is」「am」「are」のこと。
* be動詞は主語と動詞をイコールで繋ぐ。

**使い分け**
| 　　  | 主語      | be動詞   |
| ----- |-------| -----------|
|  | I     | am  |
|  | You     | are |
| 単数 | Tarou/she/he | is |
| 複数 | Tarou and Gorou/We | are |

example
```
I am an engineer.
(私は、エンジニアです。)

【I am の短縮形】
I'm an engineer.
```

```
You are engineer.
(あなたは、エンジニアです。)

【You are の短縮形】
You're engineer.
```

```
She is engineer.
(彼女は、エンジニアです。)

【She is の短縮形】
She's engineer.
```

```
We are engineer.
(私達は、エンジニアです。)

【We are の短縮形】
We're engineer.
```


### "自己紹介"

| 項目 | 日本語 | 英語 |
| ------- | ------- | ------- |
| 出身地 | 私は〜の出身です。 | I'm from ~ |
| 年齢 | 私は〜歳です。 | I'm ~ years old.|
| 職業 | 私は〜です。 | I'm ~ |
| 職場 | 私は〜で働いています。 | I work ~ |

example
```
I'm from Miyazaki.
(私は宮崎の出身です。)
```

```
I'm 26 years old.
(私は26歳です。)
```

```
I'm an engineer.
(私はエンジニアです。)
```

```
I work for a software development company.
(私はソフトウェア開発会社で働いています。)
```


### "一般動詞"
* be動詞以外のすべての動詞を一般動詞という。
* 文には動詞は一つで良いので、be動詞と一般動詞を一緒に使わないように注意する。
* これらの動詞を使うときは日本語との語順の違いに注意。
```
「誰が(主語)」→「どうする(動詞)」→「何を」
```

example
```
I play games.
(私はゲームをします。)
```

```
I study English every day.
(私は毎日英語を勉強します。)
```

```
I speak English.
(私は英語を話します。)
```

### "人称"
* 人称には1人称、2人称、3人称の3種類がある。

|人称|説明|例|
| ------- | ------- | ------- |
| 1人称 | 自分(私・私たち) | I・We |
| 2人称 | 相手(あなた・あなた達) | You |
| 3人称 | 自分と相手以外 | She・He・Tarou |

example
```
※1人称
I like baseball.
(私は野球が好きです。)
```

```
※2人称
You are kind.
(あなたは優しいです。)
```

```
※3人称
He is from Japan.
(彼は日本の出身です。)
```
### "動詞の形"
* 「そのままの形」と「sをつけた形」で使い分ける。
* 主語がIまたはYouのときは、一般動詞の形は変わらない。
* IまたはYouでもない時(つまり3人称のとき)は、動詞にsをつける(複数の場合は例外)。
* 動詞につけたsのことを「3単現のs」という。
* 主語が複数の場合は、動詞にsはつけず、そのままの形を使う。

example
```
※三人称単数は動詞にsをつける
Tarou walks to work every day.
(太郎は、毎日職場まで歩きます。)
```

```
※主語が複数の場合は動詞にsはつけない
They walk to work every day.
(彼らは、毎日職場まで歩きます。)
```
* 「3単現のs」は基本的には動詞にそのままsをつけるが、一部そうでない動詞がある。

|種別|動詞|3単現|
| ------- | ------- | ------- |
| 特別な形に変化 | have | has |
| esに変化 | go | goes |
|  | teach | teaches |
|  | watch | watches |
|  | wash | washes |
| y→iesにする動詞 | study | studies |

example
```
Tarou has a cat.
(太郎は、猫を一匹飼っています。)
```

```
He watches TV.
(彼は、テレビを観ます。)
```

```
She studies English.
(彼女は、英語を勉強します。)
```
### "代名詞"
* 代名詞とは、名詞の代わりに使われることば
* 代名詞には、「人称代名詞」、「所有代名詞」、「再帰代名詞」、「指示代名詞」、「不定代名詞」がある
* 一度話に出てきた人やモノを指したり、人を紹介したいするときによく使われる

※代名詞の例
| 単語 | 意味 |
| ------- | ------- |
| I | 私は |
| You | あなたは |
| He | 彼は |
| She | 彼女は |
| It | それは |
| We | 私達は |
| They | 彼らは・彼女らは・それらは |

example
```
This is my brother.
(こちらは、私の兄です。)

He is from Japan.
(彼は、日本の出身です。)
```

### "所有格(代名詞)"
* 所有格(代名詞)とは、人やモノの関係性について表現することば
* 代名詞を変化させて「だれだれの~」という意味を表す
* 所有格は名詞の前でしか使えない
* 名詞にaやtheをつけれはいけない
* 所有格を使わない場合は、「's」つける

※所有格の例
| 単語 | 意味 |
| ------- | ------- |
| My | 私の |
| His | 彼の |
| Her | 彼女の |
| Your | あなたの |
| Our | 私達の |
| Their | 彼らの・彼女らの |

example
```
※所有格を使う場合
This is my pen.
(これは、私のペンです。)

※所有格を使わない場合
Taro's pen.
(太郎のペンです。)
```
